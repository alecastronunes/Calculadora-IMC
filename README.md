## Sobre

A Calculadora de IMC é um projeto pessoal desenvolvido com o objetivo de colocar em prática os conceitos que venho estudando ao longo da minha formação em desenvolvimento web. A aplicação foi construída utilizando Vite, React com TypeScript e React Router DOM, este último responsável pela criação das rotas de navegação entre as páginas Home, Saiba Mais e NotFound, além de garantir a renderização do componente Header em todas as páginas da aplicação.

O desenvolvimento deste projeto foi extremamente relevante para o meu aprendizado, pois me permitiu aplicar, de forma prática, conceitos fundamentais do ecossistema React. Entre eles, destaco o uso do hook useState para gerenciamento de estado dos componentes, a utilização de interfaces do TypeScript para garantir tipagem segura e consistente dos dados exibidos, e a aplicação de renderização condicional, utilizada para exibir a seção de resultados apenas quando os dados necessários estivessem devidamente preenchidos.

Durante o desenvolvimento, enfrentei algumas dificuldades iniciais relacionadas à correta utilização de interfaces no TypeScript, especialmente no processo de tipagem de estados. Para superar esse desafio, revisitei minhas anotações e materiais do curso que abordavam esse conteúdo, o que me permitiu aplicar a solução correta e consolidar ainda mais o conhecimento adquirido.

De forma geral, considero este projeto de grande importância para o meu desenvolvimento como programador. Construir a aplicação do zero, de forma independente, buscando soluções nos materiais de estudo sempre que surgiam obstáculos, contribuiu significativamente para aumentar minha autonomia, confiança e capacidade de resolver problemas — habilidades essenciais para atuar em projetos futuros e em ambientes profissionais.

## Instalação

Instruções sobre como instalar e configurar o projeto.

```bash
# Clone o repositório
git clone https://github.com/alecastronunes/Calculadora-IMC.git

# Entre no diretório do projeto
cd seu-repositorio

# Instale as dependências
npm install
```

## Uso

Instruções sobre como usar o projeto após a instalação.

```bash
# Comando para iniciar o projeto
npm run dev
```
